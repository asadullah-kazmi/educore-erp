# School ERP & Finance Management System

This repository contains a scaffold for the School ERP Windows desktop application (C#, WPF, .NET Framework 4.8).

Structure:

- `src/` — application source
- `docs/` — design and architecture notes
- `installers/` — deployment artifacts and guides
- `tests/` — test plans and test code

See `school-erp-plan.md` for the detailed project plan.
